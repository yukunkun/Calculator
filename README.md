# 这是一个基于NDK的一个简单计算器的demo
#### 这里使用了java调用C代码的完成的一个计算器的功能的demo
#### 实现了+ ,- ,* ,/ 四个功能,每次只能运行一个运算,不能连续运算,运算的结果可以作为第二次的运算初始值
#### GrideView做的计算器的按键键盘,可以clear数据,可以每次清除一个数据.C语言写的+ ,- ,* ,/ 的方法进行的运算
